My Projects

Here are some projects I’ve worked on:

Aplikasi Pengelolaan Laundry

Built with Laravel, MySQL, and Power Designer (for DFD purposes).

  • Role-based access for Admin, Kasir, and Owner
  • CRUD for outlets, users, and laundry packages
  • Transaction entry with automatic report generation
  • Logging system and database triggers for activity tracking

This project taught me how to build a full Laravel CRUD app with clean structure, role management, and reliable backend logic.

Source code:
https://github.com/ShiruRino/kelola-laundry

Aplikasi Kasir Restoran (Table Service)

Built with Laravel, MySQL, and Power Designer (for DFD purposes).

  • Role-based access for Administrator, Waiter, Kasir, and Owner
  • CRUD for menu, orders, users, and transactions
  • Automatic report generation for sales and table activity
  • Database design based on PDM with stored procedures and relationships

This project taught me how to apply OOP principles, connect Laravel with a structured database, and design a clean user-friendly POS system.

Source code:
https://github.com/ShiruRino/kelola-resto

Flutter–Laravel CRUD Integration

Built with Flutter and Laravel.

  • Connected Flutter frontend to Laravel REST API
  • Implemented full CRUD (Create, Read, Update, Delete)
  • Used Dio for efficient API calling and error handling
  • Displayed real-time data with JSON response from Laravel

This project taught me how to integrate Flutter with a Laravel backend using Dio and manage data flow smoothly between both platforms.
