My Projects

Here are some projects I’ve worked on. Each one started as a small idea that turned into something bigger once I got into it.

Aplikasi Pengelolaan Laundry

This one’s a full management app I built using Laravel and MySQL.

  • Has roles for Admin, Kasir, and Owner
  • You can add and manage outlets, users, and laundry packages
  • Handles transactions and auto-generates reports
  • Includes logging and triggers to track what’s happening in the database

This project taught me how to build a clean Laravel CRUD app with proper role management and a solid backend structure.

Source code:
github.com/ShiruRino/kelola-laundry

Aplikasi Kasir Restoran (Table Service)

Another Laravel project, but this time for restaurant management.

  • Has roles for Administrator, Waiter, Kasir, and Owner
  • You can manage menus, users, orders, and transactions
  • Generates automatic reports for sales and table activity
  • Built with a strong database design using PDM and stored procedures

This one helped me understand OOP concepts better and how to make a POS system that’s easy to use and maintain.

Source code:
github.com/ShiruRino/kelola-resto

Flutter–Laravel CRUD Integration

This project was me trying to connect Flutter with Laravel for the first time. I used Flutter for the frontend and Laravel for the backend API.

  • Flutter sends requests to Laravel’s REST API
  • Full CRUD works perfectly (Create, Read, Update, Delete)
  • Used Dio to handle API calls and manage errors
  • Displays real-time JSON data between both platforms

I built both sides myself to understand how data moves from mobile to backend and back. It felt amazing when it finally worked after hours of debugging.

Source code:

  • Frontend:

  • Backend:
    github.com/ShiruRino/backend-flutter
